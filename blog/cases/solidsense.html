Solidsense

Solidsense is a innovative company that brings ideas to live - emotional surroundings, experiences, installations and interactive exhibits,
helping companies on delivering stunning experiences, participate on planning, execution and production on individual solutions to bring
their dreams into reality using technology.
We have the privilege to work with solidsense on a solution of Event Media Streaming, this platform runs on AWS, and together
we improve the availability, cost and performance of the solution.
The solution utilizes various aws services like ElasticBeanstalk, RDS, Lambda, S3, CloudFront, Cognito, Chime and other services.
We help defining a good IaC aproach using Terraform, multi-account structure, Disaster Recovery strategy, etc,
and also implement all of these best practices in the AWS Infrastructure optimizing the cost, availability and performance.

This partnership begins with an initial Architecture Review analyzing the Security, Reliability, Operational Excellence
and also the Terraform Setup and Project Structure,
SCC Brasil was contracted to improve the cloud infrastructe to follow all best practices and also the 5 pillars of the Well-Architected Framework.

Between the various upgrades made to the infrastructure, we can cite some memorables:
- IaC using Terraform, and Azure DevOps to deploy the code using Pipelines.
- Improve Auto-Scaling configuration to fit the needs of an Event (huge spikes in traffic in minutes).
- Improve security using SecretsManager, Security Group Chaining, Cloudfront Origin Access Control, etc.
- Improve Backup in a multi-account setup using AWS Backup.
