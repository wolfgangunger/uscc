Solidsense

Solidsense is a innovative company that brings ideas to live - emotional surroundings, experiences, installations and interactive exhibits,
helping companies on delivering stunning experiences, participate on planning, execution and production on individual solutions to bring
their dreams into reality using technology.
Real - Hybrid - Digital. Solidsense will support you during all project stages: Planning, Consulting, Execution and Production.
See company site for more infos: Solidsense


We had the privilege to work with solidsense on a solution of Event Media Streaming, this platform runs on AWS, and together
we improved the availability, cost and performance of the solution.
Their modern and innovative solution utilizes various aws services like ElasticBeanstalk, RDS, Lambda, S3, CloudFront, Cognito, Chime, Amazon IVS(Interactive Video Service) and more.
We helped on the IaC aproach (Terraform), re-structuring the Terraform Project to allow them to scale and therefor to deliver their platform faster to more clients.
Together with Solidsense we planned and defined a really sophisticated Backup and Disaster Recovery strategy
and also implemented all of these best practices in the AWS Infrastructure optimizing the cost, availability and performance.

This partnership begun with an initial Architecture Review analyzing the Security, Reliability, Operational Excellence
and also the Terraform Setup and Project Structure and turned in a partnership for about 10 month

Between the various upgrades made to the infrastructure, we can cite some memorables:
- Best Practices on IaC using Terraform to enable a quick rollout of the platform for new clients
- Time scheduled Auto-Scaling configuration to fit the needs of an Event (huge spikes in traffic in minutes).
- Setting up Azure Pipelines for Terraform Deployments and Application Code Builds
- Setting up Release Pipelines for Deployment of Application Code (ElasticBeanstalk & Lambdas) - Cross Region and Cross Account Backups for maximum Reliability and Availability.


We are glad to be able to support the Solidsense Team on their Cloud journey, it was a lot of fun to work with them, we want to thank in special Soeren and Lena and hope all the Best for them in the future.
